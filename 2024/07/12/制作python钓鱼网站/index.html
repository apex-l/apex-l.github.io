    制作python钓鱼网站

    第一周 python 钓鱼
    发布时间 : 2024-07-12 20:11
    字数:317 阅读 :
    1. 下载小皮面板并打开apache和mysql服务
    2. 新建一个网站
    3. 找到python下载网页并另存
    4. 一起拖入到phpstudy网页根目录下并将网页重命名为index.html
    5. 使用大灰狼或者cs生成一个木马软件(这里懒得搞用计算器.exe代替)并将python安装包一起压
    6. 打开Restorator.exe并将压缩包和python安装包一起拖进去
    7. 将python的图标复制粘贴到压缩包的图标里并将之前图标删除，然后保存
    8. 保存之后会将之前的安装包备份，这里图标还没有变，我们要将更换后的安装包拖入另一个文件夹图才会变
    9. 将压缩包重命名为python压缩包的名字然后再拖入刚才制作的python网页的根目录里
    10. 双击打开index.html并找到下载python按钮的代码所在地
    11. 用记事本打开index.html并修改下载路径并保存退出
    12. 重新打开网页并点击下载就会将我们刚刚制作好的压缩包下载到本地，只要双击，那么压缩包里的木马文件就会在后台悄悄运行，从而达到监听目标电脑的目的
