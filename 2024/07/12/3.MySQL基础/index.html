    MYSQL基础

    第一阶段 MYSQL
    发布时间 : 2024-07-12 20:30
    字数:179 阅读 :

    1. 在小皮面板打开MySQL并找到MySQL的bin目录下输入cmd打开命令行

    2. 输入mysql -u用户名 -p密码，连接数据库

    3. 使用show databases;查看所有数据库

    4. 使用create database 数据库名;，创建数据库

    5. use test;使用test数据库

    6. show tables;查看test数据库下所有表

    7. 创建用户表

    8. desc users;查看user表结构

    9. insert into插入数据，有两种插入方式

    10. select * from users;查看users表里所有内容

    11. update users set username = ‘admin’,password=’7777’ where username=’lisi’;修改表数据

    12. delete from users where id=1;删除表数据

    13. drop table 表名;drop database 数据库
