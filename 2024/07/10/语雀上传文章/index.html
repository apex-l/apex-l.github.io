    语雀上传文章

    第一周 语雀 文章
    发布时间 : 2024-07-10 19:11
    字数:158 阅读 :
    1. 选择一篇文章公开
    2. 点击文章右边三个点在点击导出
    3. 选择以markdown形式导出，然后在下一页选择导出 LaTeX 公式为 Markdown 语法
    4. 导出到blog文件夹下的source文件夹下的_posts文件夹里面
    5. 双击刚刚导出文件以vscode打开并添加标题

    1. 因为语雀有防盗链，导致文件里的图片无法在博客上显示所以我们要在D:\tools\lgit\blog\themes\3-hexo\layout_partial目录下找到header.jes文件并添加即可
    2. 操作完毕后在blog目录下打开git bash here,输入hexo d -g上传并部署
