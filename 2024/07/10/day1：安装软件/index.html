    安装软件

    第一周 软件
    发布时间 : 2024-07-10 19:11
    字数:834 阅读 :

    2. 2.安装python3.9
    3. 3.安装jdk22
    4. 4.安装小皮面板
    5. 5.安装IDEA
    6. 6.安装vs code
    7. 7.激活win10系统
    8. 8.edge浏览器安装代理插件

    1.安装vmware 17

    去官网下载好安装包之后右键点击以管理员身份运行，在安装界面点击自定义安装。
    注意：设置好安装路径（不能有中文和空格），把能勾的都勾上，取消产品更新。
    安装好之后记得输入许可证

    2.安装python3.9

    去官网下载好安装包后鼠标双击运行，勾上自动添加python路径，然后点击自定义安装。在第二个页面全部打勾，第三个页面的倒数两个选项不打勾并设置好安装路径(不能有中文和空格)。
    安装好之后可以右键此电脑点击属性，打开高级系统设置，点击环境变量，在系统变量中找到Path并双击打开，查看路径是否设置成功

    然后在用户变量里找到path并双击打开，查看是否有python路径，如果没有则表示安装成功

    win+r输入cmd打开系统运行框。输入python -v查看python版本

    3.安装jdk22

    在官网找到对应的安装包并点击下载。
    下载好双击运行，设置好安装路径(不能有中文和空格)。
    找到jdk路径，右键此电脑，点击属性，在高级系统选项中打开环境变量。
    在系统变量中新建一个JAVA_HOME并把jdk路径复制进去

    找到Path双击打开并添加%JAVA_HOME%\bin

    最后win+r输入cmd打开系统运行框，输入java -version查看jdk版本

    4.安装小皮面板

    在官网找到Windows64版本并下载，右键点击以管理员身份运行打开安装面板并设置好安装路径(不能有中文和空格)。
    安装好之后启动apache和mysql

    在软件管理中下载phpMyAdmin4.8.5

    点击网站-管理-打开网站

    5.安装IDEA

    下载好安装包之后双击打开并设置好安装路径，把能打勾的都勾上。
    在文件夹全家桶激活包中双击启动破解.bat
    激活完毕，重启电脑
    在文件夹全家桶激活包中双击打开IDEA 激活码.txt并复制全部代码

    双击打开IDEA，点击Activation code并把刚刚复制的代码粘贴进去即破解完毕

    6.安装vs code

    双击打开安装包，设置好安装路径
    在工具箱中安装以下三个插件：
    1.codeium（AI写代码）
    2.chinese（编辑器语言设置成中文）
    3.code runner（运行代码）
    注意：安装codeium时需要去官网注册账号并进行登录，使用时要确认账号已登录成功，CTRL+I进行使用

    7.激活win10系统

    将Windows自带的所有杀毒软件和防火墙关闭

    将激活软件拖入系统并双击一键激活

    8.edge浏览器安装代理插件

    在edge浏览器中找到右上角三个点，点击拓展，打开微软自带的拓展安装页面，在搜索框里输入proxy switchyomega并安装第一个
    安装好之后打开拓展，点击选项-新建前景模式-创建情景模式名并点击创建，设置好代理服务器，代理协议，代理端口

    保存之后在cip.cc网站中查看ip所在地
