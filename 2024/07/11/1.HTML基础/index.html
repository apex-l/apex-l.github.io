    HTML标题

    HTML 标题（Heading）是通过h1 - h6标签来定义的。

    HTML 段落

    HTML 段落是通过标签p来定义的

    HTML 链接

    HTML 链接是通过标签a来定义的。

    HTML 图像

    HTML 图像是通过标签img来定义的.
    img是空标签，意思是说，它只包含属性，并且没有闭合标签。
    src属性用来存放图片的URL的地址或本机的存储路径
    alt 属性用来为图像定义一串预备的可替换的文本，在浏览器无法载入图像时，替换文本属性告诉读者她们失去的信息。

    HTML 表格

    HTML 表格由table标签来定义。每个表格均有若干行（由tr标签定义），每行被分割为若干单元格（由td标签定义），表格可以包含标题行（th）用于定义列的标题。

    HTMLdiv元素

    HTMLdiv元素是块级元素，它可用于组合其他 HTML 元素的容器。
    如果与 CSS 一同使用，div 元素可用于对大的内容块设置样式属性。
