    Linux更换镜像源并下载常用工具

    第一阶段 Linux
    发布时间 : 2024-07-19 17:30
    字数:437 阅读 :
    1. 1.更换镜像
      1. 备份
      2. 下载
      3. 添加EPEL
      4. 清理缓存并生成新的缓存
    2. 2.下载常用软件

    1.更换镜像

    备份

    cp /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup
    将现有的 CentOS 仓库配置文件 CentOS-Base.repo 备份为 CentOS-Base.repo.backup。

    下载

    curl -o /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
    使用 curl 命令从阿里云的镜像仓库下载 CentOS 7 的仓库配置文件，并保存到 /etc/yum.repos.d/CentOS-Base.repo。

    添加EPEL

    curl -o /etc/yum.repos.d/epel.repo http://mirrors.aliyun.com/repo/epel-7.repo
    使用 curl 命令从阿里云的镜像仓库下载 EPEL（Extra Packages for Enterprise Linux）7 的仓库配置文件，并保存为 /etc/yum.repos.d/epel.repo。

    清理缓存并生成新的缓存

    yum clean all
    清理所有 yum 的缓存，以确保使用最新的仓库配置和软件包信息。 yum makecache
    重新生成 yum 缓存，确保后续安装软件包时可以快速访问最新的软件包信息。

    2.下载常用软件

    yum install net-tools.x86_64 -y
    安装 net-tools 软件包，这个包包含一些网络工具，如 ifconfig。
    yum -y install epel-release
    安装 EPEL 仓库发布包，这样可以从 EPEL 仓库安装额外的软件包。
    yum -y install vim ： 安装 vim 文本编辑器。
    yum install unzip -y： 安装 unzip 工具，用于解压 .zip 文件。
    yum search wget -y
    搜索 wget 工具并显示相关信息。这条命令不一定会安装 wget，可能需要改为 yum install wget -y 来进行安装。
    yum install -y git ： 安装 git 版本控制系统。
    yum install docker -y ： 安装 Docker 容器运行环境。
    yum -y install python-pip
    安装 pip，Python 的包管理工具。
    yum install -y docker-compose
    安装 Docker Compose，用于定义和运行多容器 Docker 应用。
